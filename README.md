# Notes
* Very alpha.
* Does not include Docs.json intentionally.

# Using
## Requirements
* Docker
  * only tested with Docker for windows & gitbash terminals
* A copy of Update 8's `Docs.json` file

## Instructions
1. Checkout locally
2. Run `make install`
3. Copy `Docs.json` to `./data/`
4. Run `make utf8ify`
5. Run `make validate`
